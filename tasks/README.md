# Tasks

Dogfood tasks use the four-folder layout:

- `backlog/` - accepted but deferred
- `todo/` - ready to plan or implement
- `in-progress/` - active work
- `done/` - completed or closed work

Task IDs start at `001`. Findings promoted from `findings.md` keep their original finding IDs in task frontmatter and in the task body.

Until `bin/task` supports this layout, task files may be created manually using `tasks/TEMPLATE.md`.
