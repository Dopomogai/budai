# Glossary

## dogfood

Using budai's own workflow and task structure to manage improvements to the budai repo itself.

## registry

The canonical source of reusable budai roles, skills, workflows, runners, conventions, templates, and docs.

## four-folder layout

The dogfood task layout `tasks/{backlog,todo,in-progress,done}`. It differs from the public standard layout currently documented as `tasks/{open,archive}`.

## finding

An inbox entry in `findings.md` that records an observed issue or improvement idea before it is promoted into a proper task.
